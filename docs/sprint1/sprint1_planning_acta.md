## SPRINT 1

### Objetivo del sprint:

“Hacer un análisis del proyecto y montar un mínimo viable en una sola máquina.”

### Tareas a realizar en el sprint 1: 
- Identificar totes les tecnologies implicades
- Fer una llista de requisits funcionals i no funcionals.
- Instal·lar un servidor web (NGINX o Apache) en una màquina Linux.
- Assegurar que PHP pot connectar-se a MySQL
- Instal·lar i configurar PHP amb suport per a fitxers
- Copiar i adaptar els fitxers
- Assegurar que el formulari carrega imatges al directori uploads/
- Crear el directori uploads/ amb permisos adequats
- Provar que es mostren correctament les publicacions i imatges carregades.
- Documentar cada pas realitzat en fitxers Markdown dins del repositori
- Configurar autenticació per clau SSH entre la màquina i GitHub.
- Instal·lar i configurar MySQL
